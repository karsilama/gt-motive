# forms-ui

Reusable form UI components for Lab.

This library provides form controls and helpers used by the application.

## Purpose

- Expose reusable form controls.
- Keep form UI logic encapsulated and reusable.

## Structure

- `src/lib/form-text-control.ts`: form text control component.
- `src/lib/form-text-control.html`: component template.

## Inputs

- `label`: String
- `placeholder`: String
- `inputSuffix`: String
- `hint`: String

## Commands

- `nx lint forms-ui`
