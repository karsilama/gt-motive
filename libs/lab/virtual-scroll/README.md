# lab-virtual-scroll

Reusable virtual scroll component library.

This library contains a virtual scroll component and related models for efficient list rendering.

## Purpose

- Provide a reusable virtual scroll UI component.
- Support large lists with better rendering performance.

## Structure

- `src/lib/virtual-scroll.ts`: main virtual scroll component.
- `src/lib/virtual-scroll.html`: component template.
- `src/lib/virtual-scroll.model.ts`: scroll models.

## Commands

- `nx lint lab-virtual-scroll`
