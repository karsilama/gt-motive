# lab-button-ui

Reusable button components for the Lab UI library.

This library contains button primitives and a mini floating action button.

## Purpose

- Provide reusable button components.
- Allow consistent button styles across the app.

## Structure

- `src/lib/button.abstract.ts`: base button abstraction.
- `src/lib/button.ts`: button implementation.
- `src/lib/mini-fab-button.ts`: mini FAB button component.

## Commands

- `nx lint lab-button-ui`
