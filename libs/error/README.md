# error-domain

Shared error handling utilities for the application.

This library contains error interceptors and shared error helpers used across the workspace.

## Purpose

- Provide a common error interceptor implementation.
- Centralize shared error logic for features and services.
- Keep error handling reusable and decoupled from UI libraries.

## Usage

Use this library from feature or domain modules that need centralized error handling.

## Commands

- `nx lint error-domain`
- `nx test error-domain`
